git status

# 4) GitHubへプッシュ（main ブランチ）

1. 新しいファイルを追加する場合
# 状態確認
git status

# 個別ファイルをステージング
git add NewFile.js

# まとめて全部追加
git add .

# コミット
git commit -m "feat: 新しい機能用の NewFile.js を追加"

# GitHub へ反映
git push
